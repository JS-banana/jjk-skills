# Cross-Language Naming

Treat each language as a naming surface, not a character-conversion task.

## Choose a Relationship

- **Direct carryover**: use the same name in both markets.
- **Transliteration**: preserve sound; accept limited semantic content.
- **Semantic rendering**: preserve the idea; allow a different sound.
- **Phono-semantic recreation**: preserve enough sound while creating a useful
  meaning in the target language.
- **Parallel name**: create a distinct local name that expresses the same Brief.
- **No local name**: keep the original and provide only a pronunciation or
  descriptive line.

Do not assume a formal Chinese name is required for every English identity.

## Checks

For every target language:

- Verify the attested spelling, meaning, grammar, and cultural context of source
  material.
- Separate intended pronunciation from likely local readings.
- Check homophones, slang, personal names, geographic names, and established
  brands in priority markets.
- Check whether the local form signals an unintended category, gender, status,
  or emotional tone.
- Prefer a natural local image over mechanical literal translation.
- Ask for native-speaker review before a high-stakes launch; model research is
  a preflight, not cultural clearance.

## Reporting

Label the relationship explicitly. For example:

`中文音义再创作：保留部分英文声音，并赋予独立的中文含义。`

Never present a local brand story as the historical etymology of the source
name.
